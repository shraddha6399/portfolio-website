ROBOTICS PROCESS FOR EXAM NOTIFICATION

In this above way we cannot identify the broken links and also can skip links of some colleges which lead to the incomplete task. But by using this application, we don’t have to search the websites of different colleges for engineering entrance exam information we have designed an application using which we can send emails to school at once of a particular area which decrease time counsuption and complexity . This application uses real time filter which extract email. It is a site crawler that grab the information from the local web pages.This understand the value of time and identify the broken links on a web page that no longer works because the website is encountering one or more of the reasons below.
• An improper URL entered for the link by the website owner.
• The destination website removed the linked web page (causing what is known as a 404).
